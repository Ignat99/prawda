Флаги абзаца

   Абзац имеет тип,который определяет как этот абзац будет переводиться 
и форматироваться. Сейчас у нас есть следующие типы абзацев:
   1) Просто абзац
   2) Заголовок
   3) Преформатированный и непереводимый абзац
   4) Начало списка
   5) Элемент списка
   6) Конец списка
   Пользователь может сам вводить новый тип абзаца  или  сделать  новый 
тип выходного формата. Для этого надо отредактировать файл absatz.txt

Работающие кнопки
   Стрелки вправо-влево (Или правая кнопка мышки) Изменить тип абзаца.
   Enter Задать выбранный тип абзацу.
   Esc   Выйти из этого диалога.
